# AI와 함께하는 '바이브 코딩(Vibe Coding)' 활용 가이드
> 본 자료는 코딩 전문 지식이 없어도 AI의 도움을 받아 실무에 필요한 프로그램을 직접 개발하는 '바이브 코딩'의 개념과, 학교 및 업무 현장에서 활용 가능한 다양한 기술 스택(조합)을 안내하기 위한 연수 자료입니다.

---

## 1. 바이브 코딩(Vibe Coding)이란?
사용자가 복잡한 프로그래밍 문법을 모두 알지 못해도, **'어떤 느낌(Vibe)과 기능의 프로그램이 필요한지'** 자연어로 AI에게 기획과 요구사항을 명확히 전달하여 코드를 작성하게 하고, 이를 조립하여 결과물을 만들어내는 새로운 개발 방식입니다.

* **특징**: 개발의 진입 장벽을 획기적으로 낮춰주며, 실무자가 자신의 업무적 고충(Pain Point)을 해결하는 맞춤형 도구를 직접 만들 수 있게 해줍니다.

---

## 2. 우리가 경험한 스택: 웹(HTML/CSS/JS) + 구글 시트(DB)
가장 접근성이 좋고, 별도의 서버 구축 비용 없이 무료로 운영할 수 있는 강력한 조합입니다.

* **기술 구성**:
  * **프론트엔드 (화면)**: HTML (구조), CSS (디자인), JavaScript (동작 로직)
  * **백엔드/DB (데이터 저장)**: Google Apps Script + Google Spreadsheet
* **장점**: 교사나 실무자가 이미 익숙한 '구글 시트'를 데이터베이스로 사용하므로, 코드를 몰라도 저장된 데이터를 쉽게 관리하고 수정할 수 있습니다.
* **활용 아이디어**:
  1. **교내 시설/비품 대여 시스템**: 노트북, 특별실 등을 예약하고 현황을 구글 시트로 관리.
  2. **학생 상담 예약 페이지**: 학생들이 가능한 시간을 선택하여 예약하면, 교사의 구글 시트에 자동으로 기록됨.
  3. **방과후학교/동아리 수강 신청 페이지**: 선착순 모집 및 명단 관리를 자동화.

---

## 3. 또 어떤 도구와 조합으로 무엇을 만들 수 있을까?
웹페이지 형태 외에도 AI와 함께 다음과 같은 다양한 형태의 자동화 도구 및 앱을 만들 수 있습니다.

### A. Python + Streamlit (데이터 분석 및 대시보드)
* **특징**: 파이썬 코드를 매우 쉽게 시각화된 웹 대시보드로 만들어주는 프레임워크입니다. 데이터를 표나 그래프로 한눈에 보고 싶을 때 탁월합니다.
* **필요 도구**: Python, Streamlit, Pandas(데이터 분석 라이브러리)
* **활용 아이디어**:
  1. **학생 성적 및 모의고사 분석 대시보드**: 엑셀 성적표 데이터를 업로드하면 학급별 평균, 등급 분포 등을 그래프로 즉시 보여주는 프로그램.
  2. **설문조사 결과 시각화 앱**: 학교폭력 실태조사나 만족도 조사 결과를 동적인 차트로 정리하여 보고서 작성 보조.

### C. 노코드(No-Code) 앱 빌더 (Glide, Softr) + 구글 시트/Airtable
* **특징**: 코드를 단 한 줄도 작성하지 않고, 마우스 드래그 앤 드롭만으로 모바일 앱이나 웹사이트를 만드는 방법입니다. (AI에게 앱 설계 구조와 데이터베이스 모델링을 도와달라고 요청할 수 있습니다.)
* **필요 도구**: Glide Apps, AppSheet 등
* **활용 아이디어**:
  1. **모바일 학생 수첩 앱**: 학교 행사, 시간표, 교직원 연락처 등을 담은 교내 전용 모바일 앱 제작.
  2. **온라인 분실물 센터**: 습득한 분실물 사진을 올리고 찾아갈 수 있도록 하는 간단한 게시판 형태의 앱.

### D. 크롬 익스텐션 (Chrome Extension)
* **특징**: 구글 크롬 브라우저에 나만의 부가 기능을 추가하여, 웹 서핑이나 업무 처리 속도를 높이는 도구입니다.
* **필요 도구**: HTML/CSS/JavaScript
* **활용 아이디어**:
  1. **나이스(NEIS) 입력 보조 도구**: 반복적으로 클릭해야 하는 버튼이나 복사/붙여넣기 작업을 단축키 한 번으로 해결하는 매크로.
  2. **특정 웹페이지 텍스트 필터링**: 업무 시스템에서 특정 
### B. Python + Telegram/Kakao API (챗봇 및 알림 자동화)
* **특징**: 매일 반복되는 정보 전달이나 단순 질의응답을 메신저 봇으로 자동화합니다.
* **필요 도구**: Python, 메신저 API (Telegram Bot API 등)
* **활용 아이디어**:
  1. **학사 일정 및 급식 알리미 봇**: 매일 아침 정해진 시간에 오늘 급식 메뉴나 주요 학사 일정을 메신저로 자동 발송.
  2. **업무 리마인더 봇**: 교무실 내 주요 기안 마감일 등을 특정 메신저 방에 알려주는 봇.
키워드가 포함된 게시글만 형광펜으로 칠해주는 기능.

## 4. 성공적인 바이브 코딩을 위한 4가지 원칙

1. **"작게 시작해서 살을 붙이세요" (Agile 방식)**
   * 처음부터 "완벽한 출석 관리와 알림톡 기능이 있는 앱을 만들어줘"라고 하면 AI도 엉성한 코드를 줄 확률이 높습니다.
   * *예시*: "1단계: 빈 화면에 학생 이름 버튼만 만들어줘" → "2단계: 버튼을 누르면 색깔이 변하게 해줘" → "3단계: 그 결과를 구글 시트에 저장해줘" 순으로 단계별로 지시하세요.
2. **"오류는 화면 캡처나 에러 코드 그대로 복사해서 질문하세요"**
   * "작동이 안 돼", "에러가 나"라고 모호하게 말하기보다는, 개발자 도구(F12)의 빨간색 에러 메시지를 복사해서 주거나 현상 자체를 명확히 설명하세요.
3. **"UI/UX는 레퍼런스(참고 이미지)를 제공하세요"**
   * 디자인은 말로 설명하기 어렵습니다. 원하는 느낌의 캡처 화면을 첨부하거나 "토스(Toss) 앱 느낌의 깔끔한 디자인으로 해줘"라고 구체적인 비주얼 단서를 제공하세요.
4. **"어떤 도구를 써야 할지 모르겠다면, AI에게 먼저 물어보세요"**
   * "내가 엑셀에 있는 수백 명의 학생 데이터를 그래프로 그려주는 웹을 만들고 싶은데, 어떤 프로그래밍 언어와 도구를 쓰면 좋을까?"라고 기획 단계부터 조언을 구해보세요.
